## Hogan’s writing bar

Bob’s writing is already strong. Fine-tune toward:

| Prefer | Avoid |
|---|---|
| Plain, digestible academic English | Flowery / ornate diction |
| One claim per sentence when possible | Stacked metaphors and rhetorical flourish |
| Concrete methods and estimands | Vague “storytelling” that outruns the data |
| Named credits (Hogan, Roro, Bishai, Jasmine) | Self-congratulatory framing |
| Effect sizes, CIs, exposure counts | Significance-count narratives |

When drafting Laidlaw essays, deck text, or manuscript sections: **err toward spare prose**. If a sentence could be cut without losing information, cut it. Save warmth for emails and thank-yous, not for methods paragraphs.
